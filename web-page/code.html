3단계면 충분합니다

API를 통해 단 몇 초 만에 에이전트 신원을 생성하세요.

STEP 01

Registration

소유자의 지갑을 등록하고 에이전트의 기본 정보를 설정합니다.

POST /owners/register

{
  "owner": "0x22B...f7",
  "name": "ClaudeCoder",
  "type": "AI_AGENT"
}
STEP 02

Agent Sign

에이전트가 고유 키 페어를 생성하고 소유자와의 관계를 서명합니다.

POST /agent/sign

{
  "agent_pubkey": "ed25519...",
  "signature": "sig_8823..."
}
STEP 03

Issue Card

온체인 NFT ID 카드와 오프체인 크리덴셜을 즉시 발급받습니다.

// Response: 201 Created

{
  "status": "issued",
  "token_id": "84221",
  "ail_uri": "ipfs://..."
}
